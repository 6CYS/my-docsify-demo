# 读书笔记

> 阅读记录与心得体会

## 技术书籍

### 《JavaScript 高级程序设计》

**阅读时间：** 2025年12月

**核心收获：**
- 深入理解 JavaScript 原型链和继承机制
- 掌握闭包的实际应用场景
- 了解事件循环和异步编程

**关键笔记：**
> JavaScript 是一门基于原型的语言，理解原型链是掌握 JavaScript 的关键。

### 《Clean Code》

**阅读时间：** 2025年11月

**核心收获：**
- 代码可读性的重要性
- 函数应该只做一件事
- 注释应该解释"为什么"而不是"做什么"

## 非技术书籍

### 《原子习惯》

**阅读时间：** 2026年1月

**核心观点：**
- 小习惯累积产生大改变
- 关注系统而非目标
- 身份认同驱动行为改变

**实践应用：**
1. 每天阅读技术文章30分钟
2. 每周写至少一篇技术博客
3. 每月学习一个新技能
