# 工具推荐

> 提升开发效率的工具合集

## 编辑器与 IDE

### VS Code

最受欢迎的代码编辑器，丰富的插件生态。

**推荐插件：**
- Prettier - 代码格式化
- ESLint - 代码检查
- GitLens - Git 增强
- Markdown All in One - Markdown 支持

### JetBrains 系列

- WebStorm - JavaScript IDE
- PyCharm - Python IDE
- IntelliJ IDEA - Java IDE

## 版本控制

### Git

```bash
# 常用命令
git init                        # 初始化仓库
git add .                       # 添加所有文件
git commit -m "message"         # 提交更改
git push origin main            # 推送到远程
git pull origin main            # 拉取更新
git status                      # 查看状态
git log --oneline               # 查看提交历史
```

## 命令行工具

### 终端增强

- **oh-my-zsh** - Zsh 配置框架
- **Homebrew** - macOS 包管理器
- **nvm** - Node.js 版本管理
- **pyenv** - Python 版本管理

### 实用命令

```bash
# 文件操作
ls -la                          # 列出文件
cp -r source dest               # 复制目录
mv oldname newname              # 重命名
rm -rf directory                # 删除目录（谨慎！）

# 系统监控
top                             # 进程监控
df -h                           # 磁盘空间
free -m                         # 内存使用
```
