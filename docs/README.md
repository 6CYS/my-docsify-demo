# 欢迎来到我的文档站

> 记录学习、分享知识、沉淀思考

这里是我使用 **Docsify** 搭建的个人文档站点，用于整理和分享我的技术笔记、学习心得和生活随笔。

## ✨ 特点

- 📝 使用 Markdown 编写，简单直观
- 🔍 内置全文搜索功能
- 📱 响应式设计，支持移动端
- 🎨 支持代码高亮和复制
- 🚀 通过 GitHub Pages 免费托管

## 🚀 快速开始

1. 浏览左侧侧边栏查看文档分类
2. 使用顶部搜索框查找内容
3. 点击右上角 GitHub 图标查看源码

## 📚 内容分类

- **技术文章** - 前端、后端、工具等各类技术笔记
- **生活随笔** - 旅行、读书、思考等生活记录
