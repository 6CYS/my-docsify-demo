# 关于本站

## 这是什么？

这是我的个人知识库和文档站点，使用 [Docsify](https://docsify.js.org) 构建，托管在 [GitHub Pages](https://pages.github.com) 上。

## 为什么搭建这个站点？

1. **知识整理** - 将零散的学习笔记系统化
2. **知识分享** - 让他人也能从我的经验中受益
3. **持续学习** - 写作是最好的学习方式
4. **个人品牌** - 建立技术影响力

## 技术栈

- **Docsify** - 轻量级文档站点生成器
- **GitHub Pages** - 免费静态网站托管
- **Markdown** - 简洁的标记语言
